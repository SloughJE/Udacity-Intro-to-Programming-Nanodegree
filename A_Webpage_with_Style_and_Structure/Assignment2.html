Lesson 1.1: The Basics

Nanodegree Ideas

  • Procedural Thinking
    • creating perfectly clear and unambiguous instructions for a computer to follow
  • Abstract Thinking
    • abstracting ideas from concrete notions, e.g. 1 apple plus 1 apple = two apples to 1 + 1 = 2
  • Debugging
    • sytematically identify problems and find potential causes
  • Technological Empathy
    • understanding how the computer sees things
    • your mother does not have this
  • Systems Thinking
    • break a big problem down into smaller pieces

Definitions

  • HTML
    • Hypertext Markup Language
    • main document type of web
  • HTTP
    • Hypertext Transfer Protocol
    • main protocal of the web
  • Servers
    • computers that host the files that make up the web
  • Internet
    • world's largest computer network
  • Browser
    • a program that runs on your computer that displays files from the web

HTML

  • Tags
    • An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag. Some tags (called "void" tags) do not require a closing tag.
    • there are many
    • Go here for a nice dictionary of them
  • Elements
    • everything within a set of opening and closing tags.
  • Attribute
    • property of an HTML element
    • href for example

Computers are Stupid

Note that computers take what you instructions you give them literally.
As in the joke, "you say 'Keep an eye on that lamb' and they (the skutters) do, they sit there for three hours and watch it burn!"

Notes

All whitespace (spaces, new lines) turn into a single space.
Use tags like br and p for a new line
Inline vs. block tags
Void tags need no closing tag

Lesson 1.2: Creating a Structured Document with HTML

Boxes

Everything in HTML is a box, even circles. An imaginary box surrounds each section or element of the webpage.
Start thinking in boxes to design a webpage.

Tree-like Structure

Webpages have a tree-like structure. I am familar with this concept from the study of decision trees.
This appears as nested boxes in the browser

Indented HTML and the Boxes

The HTML document has a LOT of indentations which can be difficult to keep track of.
The indentation corresponds to the nesting of the box.

HTML Text Editors

Like Python notebooks or Rstudio, they make writing HTML easier.
Some even automatically add the closing tabs.

Lesson 1.3: An HTML Page with Style and Structure

CSS

  • Cascading Style Sheets
    • add style to the HTML document (fonts, colors, spacing...etc
    • can be added in the head of the HTML document or in a separate CSS file
    • must be in same folder

Here are some examples of code to add style to the HTML document:

The code: h1 { text-align: center; color: #294860; } makes all h1 headers centered and of the hexidecimal color #294860.

The code: <div class="lesson"> references the style rule which corresponds to this class.

Here is an example corresponding style rule which would go in the style section or file:
.lesson { background-color: #0F7F9A; border-radius: 20px; color: white; padding: 10px; margin: 10px; }

There are a multitude of style rules so just look them up or use the developer tools in Chrome to fine tune the style.

Note: do not add styling inline with the HTML code. This will lead to many problems.
